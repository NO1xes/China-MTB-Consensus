# 中国山地车运动安全与公共行为共识（草案）

Safety First. Legality First. Rational Development.

---

## 项目初衷

山地车运动在中国正处于快速发展阶段。
参与人群年轻化、场地条件复杂、法律边界模糊、公共空间冲突频发。

本项目旨在：

- 以“安全”为最高原则
- 以“道德”为基本约束
- 在中华人民共和国法律框架内
- 探索山地车运动的理性、长期、协同发展路径

本文件不是宣言，不是对抗文本，不代表任何组织。
它是一份倡导“自律”的公共共识草案。

---

## 适用范围

本文所称“山地车运动”主要指：

- XC（Cross Country）
- 林道骑行
- All Mountain
- Enduro
- Downhill

其他与越野能力相关的骑行形式，如攀爬、土坡、小轮车、山地马拉松、砾石骑行等，可参照本共识相关原则进行理解与适用。

本共识重点关注：

- 越野场景下的安全风险
- 公共空间共存问题
- Trail 使用与维护相关争议
- 青少年参与与训练问题

## 文件结构

本共识采用“分层 + 分模块”结构。

### 一、四级表达层级

1. 口号版（便于记忆与传播）
2. 宣讲版（1–2分钟活动前说明）
3. 教学与培训版（结构化学习材料）
4. 法律与解释版（法律依据与风险说明）

四级内容逻辑一致，仅展开程度不同。

---

### 二、功能模块结构

为解决“原则简短”与“场景复杂”之间的矛盾，
本项目采用以下模块划分：

- Core：核心文本

- Model：风险分级与判断模型
- Scenarios：具体场景应用

其中：

原则保持稳定

场景文件可持续扩展

## 关于 Trail 建设与维护问题的立场

Trail 建设与维护问题具有现实复杂性。

本共识立场如下：

1. 未经授权修建或改造公共场地，存在法律风险与责任风险。
2. 鼓励通过合法沟通、协商与制度化路径推动场地建设。
3. 若涉及既有 Trail 的维护行为，应优先：

   - 降低安全隐患
   - 减少水土流失
   - 避免扩大地形改造范围
4. 不鼓励对抗性行为或扩大公共冲突。

本文件不对具体行为作执法判断，
而提供风险分级与理性判断框架。

## 项目定位

- 不组织骑行活动
- 不组织修路行为
- 不代表任何俱乐部或商业机构

本项目目标是：

建立一个可检索、可引用、可持续修订的
山地车安全与公共行为理性文本基础。

长期目标：

推动中国山地车运动在安全与合法框架内实现长期协同发展。
